# 使用说明

## 你会看到什么
- 上方是文件列表，显示“是否重命名（Pick）”“原文件名”和“建议的新文件名”。
- 下方是操作按钮，可以编辑、应用、检查更新、查看说明等。
- 任何重要提示都会以弹窗显示，并告诉你“为什么出现这个提示”。

## 快捷操作
- `Space`：切换当前选中条目的“是否重命名”
- `E`：编辑当前选中条目的目标文件名
- `F`：选择其他目录
- `A`：应用重命名
- `U`：检查更新
- `L`：切换语言
- `I`：查看关于
- `H`：查看帮助
- `Q`：退出

## 右键菜单（GUI）
- 在 GUI 表格里右键某条文件，可直接：
  - 打开文件（用系统默认关联应用）
  - 打开所在目录并定位该文件（在支持的平台会选中该文件）

## 我们的命名规则
- 默认格式：`书名-作者-年份`
- 作者只保留主要作者（通常是第一作者）
- 年份优先从书内信息提取
- 如果作者或年份无法确定，就不显示该段（不会写 `Unknown...`）
- 如果书名里本来就带了作者信息，会自动去重，避免重复

## 发生重名时怎么办
- 如果两个文件改名后会重名，程序会自动在后面加序号：
- 例如：`书名-作者-年份`、`书名-作者-年份-2`

## 安全说明
- 默认是预览，不会直接改名。
- 只有你点击“应用重命名”后才会执行。
- 只会改名“已勾选/已选中”的文件。
- 对 `rfc*.pdf`（如 `rfc1928.txt.pdf`）默认不勾选。
- 如果建议文件名包含可疑字符，会默认不勾选并保留原名。
- 如果文件名不需要变化，会自动跳过。

## 文件指纹索引
- 程序会在本地维护一个文本索引文件（JSON），记录：
  - 文件大小（size）
  - 最新修改时间（mtime_ns）
  - sha256
- 这个索引用来辅助识别“不同文件名但内容相同”的重复文件。

## 切换目录
- 按 `F` 打开目录选择窗口。
- 可从根目录开始展开文件夹，使用方向键导航。
- 选中目录后按回车（或点击“选择目录”）即可切换并重新扫描。
