# 使用说明

## 你会看到什么
- 上方是文件列表，显示“原文件名”和“建议的新文件名”。
- 下方是操作按钮，可以编辑、应用、检查更新、查看说明等。
- 任何重要提示都会以弹窗显示，并告诉你“为什么出现这个提示”。

## 快捷操作
- `E`：编辑当前选中条目的目标文件名
- `F`：选择其他目录
- `A`：应用重命名
- `U`：检查更新
- `L`：切换语言
- `I`：查看关于
- `H`：查看帮助
- `Q`：退出

## 我们的命名规则
- 默认格式：`书名-作者-年份`
- 作者只保留主要作者（通常是第一作者）
- 年份优先从书内信息提取
- 如果作者或年份无法确定，就不显示该段（不会写 `Unknown...`）
- 如果书名里本来就带了作者信息，会自动去重，避免重复

## 发生重名时怎么办
- 如果两个文件改名后会重名，程序会自动在后面加序号：
- 例如：`书名-作者-年份`、`书名-作者-年份-2`

## 安全说明
- 默认是预览，不会直接改名。
- 只有你点击“应用重命名”后才会执行。
- 如果文件名不需要变化，会自动跳过。

## 切换目录
- 按 `F` 打开目录选择窗口。
- 可从根目录开始展开文件夹，使用方向键导航。
- 选中目录后按回车（或点击“选择目录”）即可切换并重新扫描。
